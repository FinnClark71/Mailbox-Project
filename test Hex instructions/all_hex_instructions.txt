HEX TEST INSTRUCTION LISTINGS
BRISKI + Tinsel-Style Mailbox Project
--------------------------------------------------------------------------

Registers used as x0-x9 (ABI: x0=zero, always 0).
MMIO base address: 0x8000 (loaded via LUI x1, 8).
BRAM PASS address: 0x3FC word offset = BRAM[0xFF].
PASS condition: SW 1 to BRAM[0xFF].

MMIO Register Map (byte offsets from 0x8000):
  0x00  STATUS    [R]   bit0=tx_ready, bit1=rx_has_msg, bit2=init_done
  0x04  TX_DEST   [W]   destination core ID
  0x08  TX_LEN    [W]   message length metadata
  0x0C  TX_SEND   [W]   write 1 to send; read = last allocated slot
  0x10  RX_POP    [W/R] write 1 to pop; read = {len[7:0], slot[7:0]}
  0x14  REL_SLOT  [W]   write slot number to release refcount
